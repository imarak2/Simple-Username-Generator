# Simple Username Generator
A very simple username generator made in Python.

# How to use
Download the Username Generator.py and put it into whatever IDE you use, and just run it. That's all it is.

# Sample Output
Here is your randomly generated username: ZenoLover2456
(If your random username has 67 or 55 in it, you'll get a new message like: "Your username is so tuff!: DiddybludWater1678")
